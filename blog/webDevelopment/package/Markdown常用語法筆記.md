# Markdown個人常用語法筆記

- 大標題
```md
    # 標題內容
```
- 程式碼區塊
```md
    ```md
    程式碼區塊
    ```
```
- 連結
```md
[連結文字](連結網址  "連結title屬性值" )
```
- 圖片
```md
![圖片alt值](圖片路徑)
```
- 清單
```md
- 項目1
- 項目2
- 項目3
```
